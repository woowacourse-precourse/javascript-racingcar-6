## 기능 요구 사항

1. 게임 인터페이스
- [x] 게임 시작 시 게임 시작 문구 출력 및 자동차 이름 입력
- [x] 사용자의 시도할 횟수 입력
- [x] 각 시도 마다 실행 결과 출력
- [x] 최종 우승자 출력

2. 자동차
- [x] 랜덤 생성 후 자동차 전진
- [x] 모델 생성 후 자동차 추가

3. 게임 시스템
- [x] 반복횟수 만큼 자동차 전진 반복

4. 입력 오류 처리
- [x] 자동차 이름 입력 오류 처리
- [x] 시도 횟수 입력 오류 처리

5. 테스트 작성
- [x] 입력 오류 처리 테스트
- [x] 시도 횟수 입력 오류 처리 테스트
- [x] 실행 결과 테스트
- [x] 최종 우승자 출력 테스트
