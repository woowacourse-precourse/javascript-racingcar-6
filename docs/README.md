## 🗂 미션 - @@@ 기능 목록

###  📚 기능 목록

- 1. [ ] 게임 시작 메세지 띄우기
  * [ ] : 견주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
  * [ ] : 이름은 5자 이하만 가능
- 2. [ ] 시도할 횟수 메세지 띄우기
  * [ ] : 숫자인지 확인
- 3. [ ] 경주 시작 
  * [ ] : 각 자동차에 대해 무작위로 숫자를 생성
  * [ ] : 생성된 숫자가 4 이상인 경우에만 자동차의 위치를 1씩 증가
- 4. [ ] :  경주 결과 출력
  * [ ] : "-" 문자를 사용하여 경주 상황을 시각적으로 표시
- 5. [ ] : 우승자 띄우기
  * [ ] : 우승자는 한명 이상일 수도 있음
  * [ ] : 우승자가 여러명일 경우 쉼표로 구분
