# 자동차 경주
## 기능 flow
    - 자동차 이름 입력받기
    - 이동 횟수 입력받기
    - 자동차 시뮬레이션 및 과정 출력
    - 우승자 출력

## 함수 및 상세
    - 차량 이름 입력
        - 유효성 검사
            - 글자 길이
        - 입력값 반환
    - 시도 횟수 입력
        - 유효성 검사
            - 숫자 여부
        - 입력값 반환
    - 시뮬레이션
        - 전진 시뮬레이션
        - 각 차량에 대한 레이스
            - 시도 횟수만큼 반복
        - 우승자 출력
    - 메세지
        - 입력 받는 메세지
        - 에러 메세지
