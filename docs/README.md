# 자동차 경주

## 기능 목록 정리

- 경주할 자동차 이름 입력 받기 (이름은 5자 이하만 가능, 쉼표로 구분)

```
pobi,woni,jun
```

- 이동 시도할 횟수 입력 받기
- 전진 여부 판단: 0에서 9 사이 무작위 값을 구하고, 값이 4 이상일 경우 전진한다.
- 각 차수별 실행 결과 출력

```
pobi : --
woni : ----
jun : ---
```

- 최종 우승자 출력(우승자가 여러 명일 경우 쉼표로 구분)

```
최종 우승자 : pobi, jun
```

- 사용자가 잘못된 값을 입력한 경우 throw문으로 예외 발생시킴

```
[ERROR] 숫자가 잘못된 형식입니다.
```
