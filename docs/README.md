-   [x] 게임 관련 문구 출력

## 모듈 모음

-   [x] 사용한 모듈 하나로 만들기

## 세부 기능 구현

-   [x] 잘못된 입력값 Error 처리 및 종료
-   [x] 자동차 이름 쉼표(,)로 구분하여 5자 이하로 입력받기

-   [x] 자동차가 주어진 횟수 동안 전진 또는 멈추는 동작 구현

-   [x] 자동차 경주 게임 우승자 알려주기
