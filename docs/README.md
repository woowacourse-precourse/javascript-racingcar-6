# prologue

이번엔 mvc패턴을 참고하여 코드를 설계
여기서 view의 경우 콘솔에 출력만 하기에 제외

종합저으로
생성할 파일은

-   App.js : 메인함수(시작지점)
-   Data.js : 상수 및 변수들 저장
-   Controller.js : 데이터 조작 후 출력까지 이후 데이터 업데이트
