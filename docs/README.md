# 구현할 기능 목록

1. 경주 할 자동차 이름, 시도횟수 입력받기
2. 전진하는 조건 : 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우 구현하고 결과 출력하기
3. 전진 완료 후 누가 우승자인지 출력하기
4. 우승자가 여러명인 경우 쉼표로 구분하여 출력하기
5. 잘못된 입력값 받을 시 에러예외 발생시키고 애플리케이션 종료하기
