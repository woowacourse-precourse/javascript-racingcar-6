# 🚘 자동차 경주 🚘

## ✔︎ 기능 목록
* [E] : 예외 처리 -> throw 문 사용

      ex) [ERROR] 숫자가 잘못된 형식입니다.

### (1) 🙋 사용자 : 자동차 n대 이름 입력
    (i) 5자 이하의 이름 입력
    (ii) ,(쉽표)로 구분
      [E] 1자 미만 or 5자 초과 이름을 입력한 경우 -> 앱 종료
      [E] 자동차 이름이 중복되는 경우 -> 앱 종료

### (2) 🙋 사용자 : 시도 횟수 m회 입력
      [E] m이 숫자가 아닌 경우 -> 앱 종료

### (3) 📝 m회 실행결과 출력
    (i) 자동차별 랜덤 값(0 ~ 9) 중 4 이상이면 전진
    
### (4) 🏆 우승자 출력
    가장 많이 전진한 자동차 이름 출력
      -> 여러 명이면 ,(쉼표)로 구분하여 출력
