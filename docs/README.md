# 구현할 기능 목록

1. 경주 자동차 생성하기
    - 각 자동차의 이름 입력받기
    - 쉼표(,)로 각 자동차 이름 구분
    - 자동차 이름은 5자 이하여야 함(예외 시 에러문 출력)

2. 경주를 몇번 시도할지 입력받기
    - 사용자로부터 경주를 몇번 시도할지 입력받기
    - 숫자 1 이상의 수만 입력 가능함(예외 시 에러문 출력)

3. 자동차 경주 진행하기
    - 랜덤 숫자 생성(0~9 사이의 숫자)
    - 랜덤 숫자가 4보다 클 경우에 자동차 전진함
    - 사용자가 입력한 경주 시도 횟수만큼 자동차 이동 실행함

4. 결과 발표(우승자 발표)
    - 실행결과를 경주 시도 횟수에 따라 출력함
    - 경주가 끝나면 우승자 결과 발표함
    - 우승자가 여러 명일 경우에 쉼표(,)로 구분하여 다수 출력 가능함
