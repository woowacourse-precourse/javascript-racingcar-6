## 기능 요구사항 정리

초간단 자동차 경주 게임

- n 대의 자동차 전진 or 멈춤
- 각 자동차에 이름 부여 (쉼표 구분 + 5자 이하)
- 몇 번 이동할지 입력
- 전진 조건 : 0 ~ 9 사이 + 4 이상 값 (Random)
- 우승자 출력 (1명 이상 + 쉼표 구분)
- 잘못된 값 입력 : `throw`문
