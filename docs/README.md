# 기능 목록

## mission2 : 자동차 경주

### 1 : 경주에 사용할 자동자 이름을 입력받는다.

- ','로 구분되지 않거나 이름이 5글자가 넘으면 에러를 출력한다

### 2 : 진행할 횟수를 `입력받는다.

- 입력받은것이 숫자가 아니면 에러를 출력한다

### 3 : 게임하기

1. 각 자동차마다 0에서 9까지의 랜덤 수를 발생시킨다
2. 랜덤 수가 4 이상일 경우에만 전진을 한다.
3. 현재 상황을 프린트한다.
4. 1부터 2의 과정을 입력받은 진행 횟수만큼 진행한다.

### 4 : 우승자 발표하기

1. 우승자를 ', '로 구분하여 출력한다.
