## 🚀 기능 목록

### Car

- [x] 0에서 9사이 무작위 값이 4이상이면 자동차는 전진한다.
- [x] 자동차의 이동 횟수를 알 수 있다.

### Race

- [x] 경주할 자동자를 생성한다.
- [x] 주어진 횟수동안 경주를 진행한다.
  - [x] 매 차수에서 자동차는 전진하거나 멈춘다.
  - [x] 각 차수별 실행 결과를 출력한다.
  - [x] 모든 차수가 끝나면 우승자가 정해진다.

### App

- [x] 경주할 자동자 이름을 입력한다.
- [x] 자동차 이름은 1자보다 작거나 5자를 초과하면 에러 문구를 반환한다.
- [x] 시도할 횟수를 결정한다.
- [x] 경주를 실행한다.
- [x] 우승자를 출력한다.
