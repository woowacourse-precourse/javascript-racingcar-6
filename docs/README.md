## 구현 기능 목록

### 경기 전

- [x] 경기 참여자 입력
- [x] 시도 할 경기 횟수 입력

### 경기 중

- [x] 입력 받은 횟수만큼 참가자별로 무작위 값 생성
- [x] 랜덤 값에 따라 전진 여부 추가
- [x] 매 경기 결과 출력

### 경기 종료

- [x] 경기 종료 시 우승자 출력
  - [x] 우승자가 여러 명일 경우 쉼표로 구분

## 예외 사항

### 참가자 입력 시

- [x] 이름이 5자 이상 입력 시 에러 처리
- [x] 빈 문자열이 있는 경우 제외하고 참가자 추가

### 경기 횟수 입력 시

- [x] 숫자 외 다른 값 입력 시 에러 처리

## 최종 점검 사항

- [x] indent 확인
- [x] 기능 요구 사항 점검
- [x] 오타 확인
