# 자동차 경주 게임

1. 자동차 이름 입력받기
	- 쉼표로 구분, 5자 이하만
2. 몇 번의 이동을 할 것인지 입력받기
3. 자동차가 이동 가능한지 랜덤한 숫자를 받아 판별
4. 회차별 자동차 이동
5. 회차 종료 후 우승자 출력
	- 쉼표로 구분


## class

- 이름, 이동횟수 기록
- 회차마다 랜덤값 받아서 판단 후 전진
